# 01.05.05 multiplication operator

```c
#include <stdio.h>

int main(void)
{
    int number = 2 * 2;
    printf("%d\n", number);

    return 0;
}

```

## Output

```txt
4
```

## Comments *[not verified]*

To multiply two operands, use * operator.  
Don't misinterpret multiplication operator with pointers which we will discuss later.
