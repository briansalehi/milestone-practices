# 01.04.02 putchar

```c
#include <stdio.h>

int main(void)
{
    char character = 'y';
    putchar(character);
    printf("\n");

    return 0;
}

```

## Output

```txt
y
```

## Comments *[not verified]*

If you need to print a character, **putchar** can be a substitution of printf as it only works with a single character.
