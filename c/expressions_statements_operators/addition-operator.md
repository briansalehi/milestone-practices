# 01.05.02 addition operator

```c
#include <stdio.h>

int main(void)
{
    int sum = 1 + 2;
    printf("%d\n", sum);

    return 0;
}

```

## Output

```txt
3
```

## Comments *[not verified]*

Using + operator, two rvalues or lvalues can be additioned.
