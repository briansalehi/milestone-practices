# 01.05.06 division operator

```c
#include <stdio.h>

int main(void)
{
    int number = 4 / 2;
    printf("%d\n", number);

    return 0;
}

```

## Output

```txt
2
```

## Comments *[not verified]*

Dividing two numbers are handled by / operator, where the left operand is divided by the right one.
