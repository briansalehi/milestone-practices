# 01.05.03 subtraction operator

```c
#include <stdio.h>

int main(void)
{
    int difference = 3 - 2;
    printf("%d\n", difference);

    return 0;
}

```

## Output

```txt
1
```

## Comments *[not verified]*

Use - operator to subtract two operands.
