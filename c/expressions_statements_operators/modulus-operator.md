# 01.05.07 modulus operator

```c
#include <stdio.h>

int main(void)
{
    int number = 11 % 5;
    printf("%d\n", number);

    return 0;
}

```

## Output

```txt
1
```

## Comments *[not verified]*

**Modulus** operator is indicated by % sign and is a binary operator.  
In standard, modulus operator a % b, is evaluated by ** a - (a/b)*b **.
