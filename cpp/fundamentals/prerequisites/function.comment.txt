<p>The <code>main()</code> function is the function that must always exist in all C and C++ programs.</p>

<p>All other functions must be defined before <i>main function</i> either by declaration, or by implementation.</p>

<p>All functions either return an object type using <code>return</code> statement or return <code>void</code> to indicate function as a non-returning function.</p>
