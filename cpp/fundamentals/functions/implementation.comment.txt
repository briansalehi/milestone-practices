<p>The function prototype basically tells what a function is called, the list of parameters the function accepts, and the return type of the function.</p>

<p>Any function in a C++ program must be declared before the <code>main()</code> function. This can be done in two ways:</p>

<ul>
	<li>Write function declaration before <code>main()</code> and then write function definition after it</li>
	<li>Write function definition before <code>main()</code> once</li>
</ul>

<p>The actual meat and potatoes the implementation of a function is what is called the definition.</p>

<p>A function definition is always comprised of a <b>statement block</b>. A <code>return</code> statement is necessary unless the function is declared with return type <code>void</code>.</p>

<p>Calling a function is the same as invoking one. When a function declaration contains parameters, the function call needs to send arguments. Arguments are values the function requests within its parameter list.</p>
