# 01.06.08 do...while

```cxx
int main()
{
    int counter = 0;

    do {
        ++counter;
    } while (counter < 10);

    return 0;
}

```

## Output

no output provided

## Comments *[not verified]*

**do while** statement works just the way **while** does.  
But it will at least do the job once before checking the  
conditional statement.
