# 01.03.01 multiple declarations

```cxx
#include <iostream>

int main()
{
    // multiple declaration and initialization
    int multiplicant = 2, multiplier = 4;

    int result = multiplicant * multiplier;

    std::cout << result << std::endl;

    return 0;
}

```

## Output

```txt
8
```
