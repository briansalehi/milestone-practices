<p><code>std::bitset</code> is the STL class designed for handling information in bits and bit flags. <code>std::bitset</code> is not an STL container class because it cannot resize itself. This is a utility class that is optimized for working with a sequence of bits whose length is known at compile time.</p>

<p>To use class <code>std::bitset</code>, include header: <code>&lt;bitset&gt;</code></p>
