<p><code>QApplication</code> is the main part of a Qt applications.</p>

<p>Applications are either graphical, which they use <code>QApplication</code> as their main application object, or they are command line based with no graphical interface, which they use <code>QCoreApplication</code> class as their main application object.</p>

<p>For now, we will only consider applications with a graphical user interface (GUI).</p>
