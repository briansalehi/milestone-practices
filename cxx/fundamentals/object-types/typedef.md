# 01.03.11 *typedef* keyword

```cxx
typedef unsigned long positive_t;

int main()
{
    positive_t distance = 1000000;

    return 0;
}

```

## Comments *[not verified]*

Some types might look so long that it's worth making an abbreviation for them.  
You can create an alias for any type you want by usign **typedef** keyword.

It's a convention to use trailing **_t** in type names, but again, it's just a matter of your taste.
