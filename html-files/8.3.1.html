Wrapper Class

As you know there are two main types of variables in Java: 1) Primitive-types 2) Reference types
Primitive-types are categorized in 8 groups:
boolean, byte, short, int, long, float, double, char
Everything in java is an object from a class, EXCEPT primitive-types. In this case we could use wrapper classes to have primitive types as an object.
As they are objects from classes, we have handy methods within these variables. (there is no method available for variables made from primitive types).

You can see these wrappers here:

Primitive type Wrapper class
boolean Boolean
byte Byte
short Short
int Int
long Long
float Float
double Double
char Char

Example of constructing from each case is in the code section.

Source

WrapperClass.java 

    public class WrapperClass {
        public static void main(String[] args) {
            Boolean aBoolean = new Boolean(true);
    
            Byte aByte = new Byte((byte) 1);
    
            Short aShort = new Short((short) 1);
    
            Integer anInteger = new Integer(1);
    
            Long aLong = new Long(1);
    
            Float aFloat = new Float(1.0);
    
            Double aDouble = new Double(1.0);
    
            Character aCharacter = new Character('c');
    
    
            System.out.println(aBoolean);
            System.out.println(aByte);
            System.out.println(aShort);
            System.out.println(anInteger);
            System.out.println(aLong);
            System.out.println(aFloat);
            System.out.println(aDouble);
            System.out.println(aCharacter);
        }
    
    }

Execution

compile


javac WrapperClass.java WrapperClass.java

run


java -cp ./java/wrapper_class WrapperClass
