unexpand

unexpands tab to spaces, -t is required to specify number of spaces to unexpand.

Source

unexpand.sh 

    #!/usr/bin/env bash
    
    echo "    a sample string" | unexpand -t 4

Execution

run


bash ./unexpand.sh

Output


a sample string
