Functions

main function, as the name suggests is the main function that always exists in all C and C++ programs.

All other functions must be defined before main function. Either they can only be declared before main and implemented after main, or they can be implemented in one step before main function.

All functions either return an object type using return statement or not.
In case of non-returning functions, return type is void.

Source

function.cxx 

    #include <iostream>
    
    // function declaration
    void demo_function();
    
    int main()
    {
        // invoking function
        demo_function();
    
        return 0;
    }
    
    // function definition
    void demo_function()
    {
        std::cout << "this is a demo function" << std::endl;
    }

Execution

compile


g++ -g -Wall -Os -c -o function.o function.cxx

link


g++ -o function.bin function.o

run


./function.bin

Output


this is a demo function
