Everything is an object - Object creation steps

When we are creating an object using new (e.g. SampleClass sampleClass = new SampleClass(1, 2);), following steps happen step by step consecutively:
* so pay attention that constructor is not the first thing that would get called after using new keyword, actually it is the last step.
1) Static inline initialization 2) Static variables (sets them with default value) 3) static blocks
4) Inline initialization 5) variables initialization (non-static variables) (sets them with default value) 6) Non-static blocks
7) Final initialization (constructor)

Source

ObjectInit.java 

    public class ObjectInit {
        public static void main(String[] args) {
            SampleClass sampleClass = new SampleClass(1, 2);
        }
    }
SampleClass.java 

    public class SampleClass {
        int number=3;       // step 4
        static int number2 = 3;        // step 1
    
        int number3;        // step 5
        static int number4; // step 2
    
        {   // step 6
            System.out.println("Non-static block executing...");
        }
    
        static {    // step 3
            System.out.println("Static block executing...");
        }
    
        public SampleClass(int number, int number3) {   // step 7
            this.number = number;
            this.number3 = number3;
        }
    }

Execution

compile


javac ObjectInit.java SampleClass.java

run


java -cp ./java/constructor ./ObjectInit
