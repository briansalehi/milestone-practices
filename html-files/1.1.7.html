Shell Shortcuts

It would be the best experience if you can touch type on your keyboard and use your 10 fingers typing anything with least effort. There are dozens of touch typing tutorial apps out there, just use one if you don’t already know how to do it. One recommended website is Typing Club. Usually programmers and admins prefer not to raise their hands for frequent tasks. So, for example you can reach to your previous commands you typed on shell by hitting up and down arrow keys, which requires you to raise your hands and move them to the top of arrow keys which seems to be a very long distance! Instead, we can get used to using the shortcuts doing exact same thing without raising our hands, as for up arrow key we use Ctrl+P and down arrow key we use Ctrl+N. NOTE:
Before you use Ctrl+S you need to know that it causes your terminal to hang, to make your terminal continue running hit Ctrl+Q afterwards. Type ”stty -xion” to make your terminal interpreting Ctrl+S as search shortcut instead. There’s a list of shell shortcuts that saves us time and energy:
Key Combination Definition
Ctrl+P Previous command
Ctrl+N Next command
Ctrl+R Search backward (Reverse)
Ctrl+S *Search forward
Ctrl+G Cancel search
Ctrl+A Move cursor to the start
Ctrl+E Move cursor to the end
Ctrl+B Move backward within a line
Ctrl+F Move cursor forward within a line
Esc then Ctrl+B Move cursor backward a word at a time, right arrow key
Esc then Ctrl+F Move cursor forward a word at a time, left arrow key
Ctrl+D Delete character under the cursor, delete key
Ctrl+K Delete all text from the cursor to the end of the line
Ctrl+X then Backspace Delete all text from the cursor to the beginning of the life
Ctrl+T Immediately transposes the two words before and under the cursor
Esc then Ctrl+U Converts text from the cursor to the end of line the word to uppercase
Esc then Ctrl+L Converts text from the cursor to the end of line the word to lowercase
Esc then Ctrl+C Capitalizes text from the cursor to the end of line the word
Ctrl+X then Ctrl+E Launch default editor
