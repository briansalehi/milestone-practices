Getting Help

There are so many commands each having many options in the shell. How can we manage?
There are a few ways we can get help when we need to know about a command or we are looking for a command to do what we want.
Manual pages are widely used in Linux shells to give you information about the available commands.
To get help of manual pages just type man command with the investigating command in front.

Sometimes the command names themselve are forgotten.
In this case, description or closes match to what that command is can be queried by man with a -k option.

We can see that we were looking for a pager and it introduced us the pager command which we call it less.

There is also another tool for reading manuals, info.
